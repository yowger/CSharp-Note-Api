## Getting Started  

### **Clone the Repository**
```sh
git clone https://github.com/your-username/crud-note-api.git
cd crud-note-api
```

### Run Project

#### Without Docker:
```sh
dotnet run
```
This will start the API on `http://localhost:5041`.

### Scalar Service
Run Scalar:
```sh
http://localhost:5041/scalar/v1
```
