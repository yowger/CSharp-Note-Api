## Getting Started  

### **Clone the Repository**
```sh
git clone https://github.com/yowger/CSharp-Note-Api.git
cd crud-note-api
```

### Run Project

#### Without Docker:
```sh
dotnet run
```
This will start the API on `http://localhost:5127`.

### Scalar Service
Run Scalar:
```sh
http://localhost:5127/scalar/v1
```
